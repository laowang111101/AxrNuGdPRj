# 项目简介

85-基于深度学习的犬种种类检测识别系统，旨在通过先进的人工智能技术，实现对犬种的高效识别。系统核心采用yolov5算法，结合Python代码开发，具备实时图像处理能力。包含UI界面，用户友好，操作简便。该系统使用了专门的数据集进行训练，确保了对多种犬类识别的高准确率，为相关研究和应用提供了有力工具。
